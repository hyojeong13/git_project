## **Project Description**

### **❤️ 주제 : 후원과 펀딩을 이용한 아티스트와 장소 연결 시스템**

### **❤️ 팀: 너와 나의 연결고리 (총4명)**

- 팀장 : 박효정 [프로젝트 총괄, DB 구축, JAVA 기능구현]
- 팀원 : 공한별 [DB 구축, JAVA 기능구현]
- 팀원 : 양상현 [DB 구축, 팀 프로젝트 디자인]
- 팀원 : 윤민지 [DB 구축, GUI 기능구현, 차트 구현]

### **❤️ 개발 목표**
- 공간 대여 서비스를 통해 창작자, 투자자, 시민 모두 참여하여 상생하는 서비스를 구현하자

1. 펀딩 서비스
